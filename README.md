# AuroraViz

AuroraViz is a clean, modern Python visualization library with curated palettes, strong typography, and minimal ink. It wraps matplotlib with opinionated defaults so your charts look good out of the box.

## Features
- Modern theme with reduced chart ink and clear hierarchy
- Curated color palettes (categorical, sequential, diverging)
- High-level chart APIs: line, bar, area, scatter, histogram, box, violin, heatmap
- Automatic figure sizing, titles, subtitles, captions
- Export helpers with retina-quality PNG/SVG
